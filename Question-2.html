What is HTML Entities

The reserved characters known as HTML entities are used in HTML documents to denote specific things. Each of these codes has an ampersand at the beginning and a semicolon at the conclusion. After learning what HTML entities are, let's go on to learning what symbol and character entities are.

List of 5 HTML Entities

quotation mark : "

ampersand : '

apostrophe : &

less-than : <

greater-than : >
